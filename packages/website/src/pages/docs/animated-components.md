# Animated Components

**React Awesome Reveal** exports a number of ready-to-use animated components.

> By default, all animations are triggered when the content they wrap enters the viewport.

## Revealing Effects

- `<Bounce />`
- `<Fade />`
- `<Flip />`
- `<Hinge />`
- `<JackInTheBox />`
- `<Roll />`
- `<Rotate />`
- `<Slide />`
- `<Zoom />`
