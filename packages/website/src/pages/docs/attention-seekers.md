# Attention Seekers

The attention seeker effects are rendered by the `<AttentionSeeker />` component, which accept an `effect` prop with the following values:

- `"bounce"`
- `"flash"`
- `"headShake"`
- `"heartBeat"`
- `"jello"`
- `"pulse"`
- `"rubberBand"`
- `"shake"`
- `"shakeX"`
- `"shakeY"`
- `"swing"`
- `"tada"`
- `"wobble"`
